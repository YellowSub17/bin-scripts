# bin-scripts

A collection of quick scripts that are usefull to have in ~/bin


## combine-pdf.py

### useage

    python combine-pdf.py a.pdf b.pdf

created a file merged.pdf that concatenates the input pdfs.


## job-monitor

### useage

    ./job-monitor x

looks for a job in htop with name "x" and will beep when it doesn't exist


## move-mouse

### useage

    ./move-mouse

moves the mouse every 5 seconds, stops screensavers

## timer

### useage

    ./timer x

sets a timer for x minutes. also takes floating point for x


## ssh-race

### useage

    ./ssh_race x

connects to and aws race hub computer at ip x
